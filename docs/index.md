# shkit

Reference documentation for `shkit`, a bash/zsh shell library of reusable utilities.

## Modules

Sourced in dependency order:

- [console.sh](console.md) — Terminal detection, ANSI color configuration, and text utilities.
- [log.sh](log.md) — Leveled logging with pretty stderr output and logfmt file output.
- [core.sh](core.md) — Core utility functions with no external dependencies.
- [validate.sh](validate.md) — Input validation and guard functions.
- [proc.sh](proc.md) — Process execution and result-handling utilities.
- [script.sh](script.md) — Script context, diagnostics, and interactive input utilities.
- [print.sh](print.md) — Styled stdout output for human-facing script UI.
- [git.sh](git.md) — Git workflow helpers.
